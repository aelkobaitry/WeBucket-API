## Git commands:

- git checkout [branchname]
    - Switch into branch branchname
- git checkout -b [branchname]
    - Switch into new branch branchname
- git pull
    - Update branch to origin
- git status
    - See changes
- git add .
    - Add all changes to be commited
- git commit -m "message to commit with"
    - Commit changes
- git push
    - Push changes
